

# This is a Task management system project.
I developed a web application by using Spring Boot.

## Tech Stack

+ Spring Boot
+ Spring Data Jpa
+ MySQl Database
+ Thymeleaf
+ Bootstrap4


## Port


```bash
 http://localhost:8091
```
